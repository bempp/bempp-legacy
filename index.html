BEM++ by bempp

BEM++ - A HPC boundary element library

The goal of BEM++ is to create an open source high-performance boundary element library for applications in electrostatics, acoustics and computational electromagnetics. Planned features include:

  • Support for all standard kernels in 2D and 3D for Laplace, Helmholtz and Maxwell problems
  • Discretisation using hp-Galerkin techniques
  • High-Frequency formulations using nonstandard approximation spaces
  • MPI/OpenMP Parallelisation on clusters and shared memory systems, and GPU acceleration on hybrid CPU/GPU clusters
  • Fast Multipole formulations to accelerate matrix vector products
  • Full integration into Matlab and Python
  • Suitability for large applications in science and industry
The project is currently in its planning phase and will be supported by EPSRC Grant EP/I030042/1 (planned start date July 2011).
